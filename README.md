# Attendance Management System

## Overview
A comprehensive attendance management system developed for ABC Engineering College by TechNova Solutions Pvt. Ltd.

## Features
- Add and manage student records
- Store course details
- Record student attendance
- Display attendance reports
- Search students by Roll Number
- Calculate attendance percentage
- Generate defaulter list
- Maintain project documentation

## Project Structure
```
AttendanceManagementSystem/
├── app.py              # Main application
├── student.py          # Student management
├── attendance.py       # Attendance recording
├── course.py           # Course management
├── report.py           # Report generation
├── database.py         # Database operations
├── utils.py            # Utility functions
├── requirements.txt    # Dependencies
├── README.md          # Documentation
├── data/              # Data files
│   ├── students.csv
│   ├── attendance.csv
│   └── courses.csv
└── screenshots/       # Documentation screenshots
```
